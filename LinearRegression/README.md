# Machine Learning

Hands-on ML implementations following the [CodeBasics YouTube playlist](https://www.youtube.com/@codebasics). Clean notebooks with practical Python covering core algorithms and workflows.

## Topics Covered

| Topic | Description |
|---|---|
| Linear Regression | Predict continuous values, understand cost & gradient descent |
| Logistic Regression | Binary classification with probability outputs |
| Support Vector Machine | Margin-based classification with kernel tricks |
| Decision Trees | Rule-based splitting for classification and regression |
| Random Forest | Ensemble method using multiple decision trees |
| Model Evaluation | Accuracy, precision, recall, F1, confusion matrix |
| Train / Test Workflows | Data splitting, cross-validation, overfitting prevention |

## Getting Started

### Prerequisites

```bash
pip install numpy pandas scikit-learn matplotlib seaborn jupyter
```

### Run Notebooks

```bash
git clone https://github.com/HJeandedieu/Machine-Learning.git
cd Machine-Learning
jupyter notebook
```

## Tech Stack

- **Python 3.x**
- **scikit-learn** — ML algorithms
- **pandas / NumPy** — data manipulation
- **Matplotlib / Seaborn** — visualization
- **Jupyter Notebook** — interactive exploration
